# customer-churn-prediction 
# 📌 Project Overview
This is a Machine Learning project based on Customer Churn Prediction in a telecom company.

The main objective of this project is to predict whether a customer will leave the company or not.

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 🤖 Machine Learning Algorithm

- Logistic Regression

---

## 📂 Dataset

Telco Customer Churn Dataset

The dataset contains:
- Customer details
- Internet services
- Contract information
- Monthly charges
- Churn status

---

## 📊 Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Visualization
5. Feature Encoding
6. Model Training
7. Prediction
8. Accuracy Evaluation

---

## 📈 Output

- Customer churn prediction
- Confusion matrix
- Accuracy score
- Graph analysis

Accuracy achieved:
- Around 80%
